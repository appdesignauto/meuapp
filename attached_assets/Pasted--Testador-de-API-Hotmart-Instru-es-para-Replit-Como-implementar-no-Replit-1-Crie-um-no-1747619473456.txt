# Testador de API Hotmart - Instruções para Replit

## Como implementar no Replit

1. Crie um novo projeto no Replit do tipo "HTML, CSS, JS"
2. Substitua todo o conteúdo do arquivo `index.html` pelo código HTML fornecido
3. Crie um novo arquivo chamado `script.js` e cole o código JavaScript fornecido
4. Execute o projeto clicando no botão "Run"

## Estrutura do projeto

```
/
├── index.html      # Contém a estrutura HTML e estilos CSS
└── script.js       # Contém toda a lógica JavaScript
```

## Como funciona

Este testador de API Hotmart permite:

1. Inserir suas credenciais de API (Client ID e Client Secret)
2. Gerar um token Basic de autenticação (codificação Base64)
3. Testar se as credenciais são válidas com a API da Hotmart

## Importante para uso real

Para um ambiente de produção, você precisaria fazer algumas modificações:

1. **Implementar a chamada real à API**: O código atual contém uma simulação. Para uma implementação real, descomente o código da função `realApiCall` no arquivo `script.js`.

2. **Segurança**: Em uma aplicação real, você não deve expor seu Client Secret no frontend. O ideal é fazer essa chamada no backend para proteger suas credenciais.

3. **Implementação de CORS**: Se você estiver tendo problemas com CORS ao chamar a API da Hotmart, você pode precisar de um proxy ou implementar essa lógica no backend.

## Exemplo de implementação real (backend)

Para uma implementação mais segura, considere criar um backend com Node.js:

```javascript
// Exemplo de código backend (Node.js com Express)
const express = require('express');
const axios = require('axios');
const app = express();

app.use(express.json());

app.post('/test-hotmart-api', async (req, res) => {
  const { clientId, clientSecret, environment } = req.body;
  
  const baseUrl = environment === 'prod'
    ? 'https://api-hot-connect.hotmart.com'
    : 'https://api-hot-connect-sandbox.hotmart.com';
  
  try {
    const response = await axios.post(`${baseUrl}/oauth/token`, 
      'grant_type=client_credentials', 
      {
        headers: {
          'Authorization': `Basic ${Buffer.from(`${clientId}:${clientSecret}`).toString('base64')}`,
          'Content-Type': 'application/x-www-form-urlencoded'
        }
      }
    );
    
    res.json(response.data);
  } catch (error) {
    res.status(400).json({ 
      error: 'Falha na validação',
      message: error.response?.data || error.message
    });
  }
});

app.listen(3000, () => {
  console.log('Servidor rodando na porta 3000');
});
```
